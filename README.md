## About

A modern, high-performance personal portfolio website built with vanilla JavaScript and Vite. Features a cinematic starfield canvas background, smooth scroll animations, 3D tilt effects, and a responsive design that works seamlessly across all devices.

## Tech Stack

| Category | Technologies |
|----------|-------------|
| **Build Tool** | Vite 6.3 |
| **Frontend** | HTML5, CSS3, Vanilla JavaScript (ES Modules) |
| **Animations** | GSAP, AOS (Animate On Scroll), Custom Canvas |
| **Typography** | Typed.js for hero typewriter effect |
| **Fonts** | Syne (headings), DM Sans (body) |
| **Deployment** | Vercel |

## Features

- **Starfield Canvas Background** вЂ” Interactive particle animation rendered on `<canvas>`
- **Typewriter Effect** вЂ” Dynamic role rotation in the hero section
- **3D Tilt Cards** вЂ” Project cards with mouse-tracking tilt interaction
- **Scroll Animations** вЂ” AOS-powered reveal animations throughout
- **Project Filtering** вЂ” Filter projects by category (All / Frontend / Full Stack)
- **Project Modal** вЂ” Detailed project view with features and live site link
- **Contact Form** вЂ” Validated form with success state
- **Mobile Responsive** вЂ” Hamburger menu and fully adaptive layout
- **Back to Top** вЂ” Smooth scroll-to-top button
- **Lightweight** вЂ” No heavy frameworks, pure vanilla JS

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Deepak2427/Portfilio_deepak.git

# Navigate to the project directory
cd Portfilio_deepak

# Install dependencies
npm install
```

### Development

```bash
npm run dev
```

Opens the portfolio at `http://localhost:5173` with hot module replacement.

### Build

```bash
npm run build
```

Generates a production-optimized build in the `dist/` directory.

### Preview

```bash
npm run preview
```

Preview the production build locally.
